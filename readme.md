# Online Shopping Clone
### **Getting started**
### Technologies Used
#### HTML5, CSS3, Bootstrap 3.3, JavaScript ES5
 - Run :  `git clone https://github.com/getwebem/Alpha-CMS.git`
 - Run :  `cd Alpha-CMS`
 - Run :  `Brackets .`
 - Open:  `Index.html`
 - Open:  `Live Preview`
